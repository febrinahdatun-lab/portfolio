Portfolio Project #5 — Dicoding BMLP

Machine Learning: Fraud Detection

Proyek Machine Learning end-to-end: dari Unsupervised Clustering (K-Means) hingga Supervised Classification (Random Forest).

Business Brief

Detecting Financial Anomalies & Transactional Risks

Proyek Machine Learning end-to-end ini dirancang untuk mendeteksi transaksi finansial yang mencurigakan (Fraud) guna melindungi ekosistem transaksi digital dari kerugian finansial. Implementasi menggabungkan metode Unsupervised Learning (K-Means Clustering) untuk memetakan kluster anomali tanpa label, dan Supervised Learning (Random Forest Classifier) untuk melakukan prediksi klasifikasi fraud secara presisi berdasarkan fitur riwayat transaksi konsumen. Dilengkapi dengan simulator penguji parameter interaktif.

Technologies Used
Python (Scikit-Learn) Random Forest K-Means Clustering Pandas & NumPy Chart.js
Accuracy: 96.02%
Approach: Clustering & RF
Node: BMLP.dataset
Status: Initializing...

🤖 Apa itu Machine Learning?

Machine Learning adalah cabang dari AI dimana komputer belajar dari data tanpa diprogram secara eksplisit. Bayangkan mengajari komputer mengenali pola, seperti mengajari anak kecil membedakan kucing dan anjing dari contoh gambar.

Analogi Sederhana

📧 Bayangkan email spam filter: bukan programmer yang menulis aturan satu per satu, tapi komputer mempelajari ribuan contoh email spam vs normal, lalu otomatis bisa mendeteksi spam baru.

🔍 Tentang Proyek Ini

Proyek ini mendeteksi transaksi fraud/anomali menggunakan 2 tahap ML:

1

Clustering (Unsupervised)

K-Means mengelompokkan transaksi menjadi 2 cluster: Normal vs Anomali

2

Classification (Supervised)

Random Forest memprediksi label dari clustering untuk data baru

📚 Konsep Kunci yang Digunakan

🎯

K-Means Clustering

Algoritma yang mengelompokkan data ke dalam K cluster berdasarkan kedekatan (jarak). Seperti mengelompokkan buah berdasarkan warna dan ukurannya.

🌲

Random Forest

Kumpulan banyak Decision Tree yang "voting" bersama. Seperti bertanya ke 100 ahli dan mengambil jawaban mayoritas.

📉

PCA (Dimensionality Reduction)

Menyederhanakan data berdimensi tinggi menjadi 2D/3D untuk visualisasi. Seperti membuat peta 2D dari dunia 3D.

⚖️

GridSearchCV

Mencoba semua kombinasi hyperparameter untuk menemukan setting terbaik. Seperti mencoba semua resep sampai menemukan yang paling enak.

⚙️ ML Pipeline — Langkah demi Langkah

1

📥 Data Collection

Mengumpulkan 2.512 sampel transaksi

Dataset terdiri dari 2.512 transaksi dengan 16 fitur: jumlah transaksi, tipe (Credit/Debit), lokasi, channel (Online/ATM/Branch), usia, pekerjaan, durasi transaksi, login attempts, dan saldo akun.

2,512 Samples 16 Features Google Sheets CSV
2

🧹 Data Preprocessing

Membersihkan & menyiapkan data

Data dibersihkan: null values dihapus, Label Encoding untuk variabel kategorikal, Standard Scaler untuk normalisasi nilai numerik agar berada di skala yang sama.

from sklearn.preprocessing import LabelEncoder, StandardScaler
3

🎯 K-Means Clustering

Mengelompokkan transaksi: Normal vs Anomali

Elbow Method digunakan untuk menentukan jumlah cluster optimal → hasilnya K=2 (Normal & Anomali). Silhouette Score digunakan untuk mengukur kualitas clustering.

KMeans(n_clusters=2, random_state=42)
4

📐 PCA Visualization

Mereduksi dimensi untuk visualisasi 2D

Data berdimensi tinggi (10+ fitur) direduksi menjadi 2 komponen utama menggunakan PCA, sehingga bisa divisualisasikan dalam scatter plot 2D untuk melihat pemisahan cluster.

5

🌲 Classification Models

Decision Tree → Random Forest → Tuned RF

Label dari clustering digunakan sebagai target supervised learning. Tiga model dilatih dan dibandingkan: Decision Tree (baseline), Random Forest (ensemble), dan Tuned Random Forest (GridSearchCV optimization). Data di-split 80:20 dengan stratifikasi.

train_test_split(X, y, test_size=0.2, stratify=y)
6

📊 Evaluation & Deployment

Precision, Recall, F1-Score comparison

Setiap model dievaluasi menggunakan Classification Report (Precision, Recall, F1-Score). Model terbaik disimpan dalam format .h5 (joblib) untuk deployment di production.
